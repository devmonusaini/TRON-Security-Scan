# TRON Wallet Security Scan Platform

A unique red cybersecurity-themed Web3 application for USDT (TRC20) security verification on the Tron blockchain.

## 🎨 Design Philosophy

This platform features a **completely unique** design approach:
- **Dark + RED cyber security theme** (NO typical Tron green/blue colors)
- Inspired by hacker consoles and threat detection systems
- Sharp edges, grid effects, and scanning animations
- Terminal-style UI with aggressive, security-focused aesthetics

## 🎯 Color Palette

- Primary Red: `#ff1a1a`
- Deep Red: `#990000`
- Background: `#050505` (almost black)
- Accent: `#ff4d4d`
- Text: `#e6e6e6`

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ installed
- TronLink wallet extension (for full functionality)

### Installation

\`\`\`bash
# Install dependencies
npm install

# Start development server
npm run dev
\`\`\`

### Usage

1. Click "INITIATE SCAN" on the hero section
2. Connect your TronLink wallet (or use demo mode)
3. Wait for the automated security scan
4. Review your security report and risk assessment
